IV. Research Questions

  • What are the most common challenges that must be addressed to adopt Big Data as solution in enterprise architecture context?.

  • How to define scope and projection to Big Data adoption in Enterprise Architecture context according to the organization type that implements it?.

  • How can Big Data technologies be implemented based on an Enterprise Architecture context?.

V. Main Goal

Develop a methodological framework that provides guidelines for the adoption of Big Data in organizations that implement Enterprise Architecture context.

VII. Expected Contributions

The expected results that have been taken as goals in the development of current proposal, include the following:
  • Suggestion on methodology in order to integrate Big Data into the proposed road map under the enterprise architecture.

  • Recommendations on Big Data usage inside Enterprise Architecture context, describing actions to be taken to obtain advantages and benefits in the organization.

  • The application of at least one case study that serves as a guide for implementation and feedback of the suggested methodology.

  • Summary of examples that methodologically adopt Big Data under a Enterprise Architecture context in organizations or state agencies.
